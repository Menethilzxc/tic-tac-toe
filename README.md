# ❌⭕ Крестики-нолики

Проект реализован на React. Играют два пользователя поочерёдно на одном экране.

##  Стек технологий

-  React
-  Redux (без Redux Toolkit)
-  JavaScript
-  Vite

##  Функциональность

- Игра 1 на 1
- Показ победителя
- Определение ничьей
- Кнопка сброса игры
- Состояние игры хранится в Redux

##  Установка и запуск

- git clone https://github.com/Menethilzxc/tic-tac-toe.git
- cd tic-tac-toe
- npm install
- npm run dev
